## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client (customer JWT) → /orders/** | Authenticated user may only cancel their own order |
| client (staff/ADMIN JWT) → /admin/orders/** | Role-gated any-order cancel |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-18-05-01 | Elevation of Privilege | plain USER reaching /admin cancel routes | mitigate | Existing `/admin/orders/**` `hasAnyRole("ADMIN","STAFF")` matcher; integration test asserts 403/401 for a USER |
| T-18-05-02 | Info Disclosure (IDOR) | customer cancelling another user's order | mitigate | Customer endpoints pass `principal.getId()`; service resolves via `findByIdAndUserId` → 404; integration test asserts 404 for non-owner |
| T-18-05-03 | Input Validation | malformed path variables | mitigate | `orderId`/`lineId` bound as UUID by Spring MVC (400 on malformed); optional reason string length-capped like `rejectionReason` |
| T-18-05-SC | Tampering | dependency installs | accept | No new dependencies |
</threat_model>
